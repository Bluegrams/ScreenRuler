Screen Ruler - Справка

Горячие клавиши

ПробелПереключить режим линейки на горизонтальный, вертикальный или двумерный.
ZВыбрать окно для измерения. Отменяется с помощью 'Esc'.
Клавиши навигации (стрелки)Переместить линейку на один пиксель.
Shift + СтрелкаПереместить линейку на один средний шаг (по умолчанию 5px).
Ctrl + СтрелкаИзменить размер линейки на один пиксель.
Ctrl + Shift + СтрелкаИзменить размер линейки на один средний шаг (по умолчанию 5px).
Alt + СтрелкаПрикрепить линейку к границам экрана.
Ctrl + CСкопировать текущую длину в буфер обмена.
LУстановить маркер на текущую длину.
CУдалить первую пользовательскую маркировочную линию.
EscВыйти из Screen Ruler, выбор окна или справка.
Дополнительные горячие клавиши в контекстном меню.

Действия мышью

Нажатие на позицию, где установлен маркер.Открыть диалоговое окно для просмотра/удаления маркера.
Двойной щелчок на линейкуУстановить маркер на выбранную позицию
Колесико мышиИзменить размер линейки.
Shift + Колесико мышиБыстрое изменение размера линейки (с большим шагом).
